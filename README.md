# ⚡ Electric Vehicles Analysis — Power BI Dashboard

A Power BI dashboard analyzing the **Electric Vehicle Population** across the United States, covering key metrics like vehicle types, model adoption trends, geographic distribution, and CAFV eligibility.

> 🎯 This is my first Power BI + GitHub project!

\---

## 📊 Dashboard Overview

The report is built on a single interactive page with the following visuals:

|Visual Type|What It Shows|
|-|-|
|📦 **KPI Cards**|Total Vehicles, Avg Electric Range, BEV Count \& %, PHEV Count \& %|
|🍩 **Donut Charts**|Vehicle breakdown by EV Type \& CAFV Eligibility|
|📈 **Area Chart**|Total Vehicles by Model Year (growth trend)|
|🗺️ **Shape Map**|Vehicle distribution across US States|
|📊 **Bar Chart**|Top vehicle Models by count|
|🌳 **Treemap**|Vehicles by Make/Brand|
|🔽 **Slicers**|Filter by City, Electric Utility, EV Type|

## 📌 Key Metrics Tracked

* **Total Vehicles** — Overall count of registered EVs
* **BEV vs PHEV** — Split between Battery Electric Vehicles and Plug-in Hybrid Electric Vehicles
* **Avg Electric Range** — Average range across all vehicles in the dataset
* **CAFV Eligibility** — Clean Alternative Fuel Vehicle eligibility status
* **Growth by Model Year** — Year-over-year EV adoption trend
* **State-wise Distribution** — Geographic spread across the US
